About Walley


Walley Language is developed by "shd101wyy".
It is a simple script language that is the
basis of the future, real, AI language..
For sourcecode, go to sourceforge.
Walley Language now have two versions.
One C language Version and One JavaScript version.

Walley Language Simple Introduction

Variable Type:

  1. int 4
  2. double 1.2
  3. fraction 4/3
  4. list []
  6. string "Hello" or 'Hello'
  7. dictionary {}
  8. class
Support:
  1. class class a: or class a extends b:
  2. function def Hi(param1,param2):
Basic:

  1. if:
    code here
    elif:
    code here
    else:
    code here

  2. while i<10:
    code here

  3. for i in range(0,6):
    code here
Attention:
  1. file io is not included in JavaScript Version.
  2. system operation is not included in JavaScript version.
